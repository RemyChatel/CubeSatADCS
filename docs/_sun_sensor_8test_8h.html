Description

This library provides a test function for the SunSensor module and also serves as an example program for the module.

To use it, simply connect three photodiodes to the Analog pins of the board.

See also
SunSensor.h

Dependencies and data type

This library depends on:

  • the Mbed framework (https://www.mbed.com/en/)
Attention
This library uses float (32-bits) and not double (64-bits) to make best use of the Floating Point Unit of 32-bits microcontrollers.
