Description

This library provide an interface to read the analog output of a set of three photodiodes that make a Sun sensor

See also
SunSensor

Example code

#include "mbed.h"
#include "SunSensor.h"
SunSensor sun();
float r_sun[3] = {0,0,0};
while(1){
sun.getSunVector(r_sun);
wait_ms(100);
}
