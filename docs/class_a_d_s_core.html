Detailed Description

Accesses, processes and filters the data from sensors to output an attitude quaternion of the Spacecraft.

Description

In order to determine the attitude (the rotation between the Earth Centered Inertial frame and the body frame), sensors are used to get a set of measurements in the body frame while models are used to provide the same measurements in the ECI frame.

The QuEst algorithm is used to find the best rotation quaternion that links each pair of body-ECI vectors.

Then a Kalman Filter is applied to the output quaterion to filter out any noise according to the dynamic model of the satellite.
