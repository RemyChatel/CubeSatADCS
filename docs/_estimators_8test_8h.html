Description

This library provides a test function for the Estimators module and also serves as an example program for the module.

Tests the Estimators module by generating a set of five model vectors and a theoretical rotation matrix. The rotation matrix is applied to the vectors in order to generate fake measurements. The model-measurement vector pair are then fed to the QuEst algorithm and the output error is computed.

See also
Estimators.h

Dependencies and data type

This library depends on:

  • the Mbed framework (https://www.mbed.com/en/)
  • <std::cmath> in order to perform cos, sin and sqrt operations.
  • <std::vector> for the matrix algebra
  • "Matrix.h" for matrices implementation
Attention
This library uses float (32-bits) and not double (64-bits) to make best use of the Floating Point Unit of 32-bits microcontrollers.
